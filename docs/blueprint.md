# **App Name**: VoiceShield AI

## Core Features:

- Dashboard UI: Display app title and description in a clean layout.
- Left Navigation Sidebar: Provide navigation options: Dashboard, Live Detection, Scam History, Settings.
- Live Detection Page Layout: Structure page with Caller Panel, Receiver Panel, Risk Score meter, AI Analysis box, and Live Transcript box.
- Scam History Page: Set up a placeholder table for scam call history.
- Settings Page Layout: Create placeholder options within the settings page.

## Style Guidelines:

- Primary color: Dark blue (#0A192F) to establish a premium cybersecurity feel.
- Background color: Desaturated dark blue (#172A3A) for a cohesive dark theme.
- Accent color: Cyan (#3DDAD7) to highlight interactive elements and risk indicators.
- Body and headline font: 'Inter' (sans-serif) for a modern, objective, and neutral appearance.
- Implement a responsive design for consistent appearance across devices.
- Use minimalist, line-style icons to maintain a clean, technical aesthetic.
- Incorporate subtle transitions for loading indicators to enhance user experience.
